# Model Simplification 
## Mesh Simplification 
Unity project for implementing Vertex-Clustering model simplification method.

## Description
This directory contains the source code of a Unity project in which I developed a script to implement Vertex-Clustering algorithm.
The script allows you to configure the tolerance of the simplification A.K.A the size of the cubes of the grid.
